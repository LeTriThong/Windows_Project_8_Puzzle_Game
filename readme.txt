Lưu ý:
- Game không thể lưu ảnh mặc định có ở đầu trò chơi, nếu muốn lưu ảnh đó thì có thể tìm ảnh sample.jpg
- Không nên kéo thả hình quá nhanh vì lệnh snap ảnh không thể thực hiện kịp
- Game có bắt một số lỗi như chọn file không phải hình ảnh, kéo ảnh ra khỏi màn hình ứng dụng,...
..................................
Các chức năng:
- Có thể tải lên bất cứ hình nào, vẫn giữ được tỷ lệ hình.
- Làm mới hình đang chơi
- Chơi game bằng hai cách: Sử dụng mũi tên và kéo thả
- Xử lý thắng/thua: Nếu hoàn thành hình trước khi hết thời gian là thắng, nếu như hết thời gian là thua 
- Lưu lại game đang chơi
- Tải lại game đã lưu, khi đã tải lại game thành công, nhấn Continue để tiếp tục
..................................
Các nút bấm:
- New game: Chọn hình mới
- Refresh: Xáo trộn hình, đặt lại thời gian
- Start: Bắt đầu chơi
- Save: Lưu game đang chơi
- Load: Tải game đã lưu
